# Raunak Varma — Portfolio

Single-page **React 18 + Vite + Tailwind CSS** portfolio with **Framer Motion** animations — *Neural Noir* aesthetic (dark editorial, cyan/teal accents).

## Stack

- React 18, Vite 8
- Tailwind CSS 3
- Framer Motion
- @tsparticles/react (neural particle background)
- react-icons, react-intersection-observer

## Scripts

```bash
npm install
npm run dev      # local dev server
npm run build    # production build → dist/
npm run preview  # preview production build
```

## Deploy

Build output is in `dist/`. Host on GitHub Pages, Vercel, Netlify, or any static host.
